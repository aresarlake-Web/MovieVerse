# MovieVerse

A modern, minimalist movie discovery web application inspired by Apple, Stripe, and Airbnb.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. (Optional) Set `TMDB_API_KEY` in a `.env` file to your TMDB API key to pull live movie data. Without it, the app uses bundled mock data.
3. Run the app:
   `npm run dev`
